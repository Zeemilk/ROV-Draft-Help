# Arena of Valor Draft Helper

แอปพลิเคชันช่วยในการเลือกฮีโร่และวิเคราะห์ทีมสำหรับเกม Arena of Valor (ROV)

## ฟีเจอร์หลัก

### 🎯 การเลือกฮีโร่
- แสดงรายการฮีโร่ทั้งหมดพร้อมรูปภาพ
- ค้นหาฮีโร่ด้วยชื่อ
- กรองฮีโร่ตาม Lane, Tier, และ Ability
- แสดงฮีโร่ที่ตอบโต้ได้เมื่อคลิกที่ฮีโร่ที่เลือกแล้ว

### 📊 การวิเคราะห์ทีม
- คำนวณสถิติทีมแบบเรียลไทม์
- แสดงข้อมูล CC, Damage, Durability, และ Time
- วิเคราะห์ประเภทดาเมจของทีม
- แสดงคำเตือนเมื่อทีมไม่สมดุล

### 📱 Cross-Platform Support
- **Responsive Design**: รองรับทุกขนาดหน้าจอตั้งแต่ 320px ถึง 1400px+
- **Touch-Friendly**: ปุ่มและองค์ประกอบขนาดเหมาะสมสำหรับการสัมผัส
- **Mobile Optimized**: Layout ที่ปรับให้เหมาะกับมือถือและแท็บเล็ต
- **Desktop Enhanced**: ประสบการณ์ที่ดีบนเดสก์ท็อป
- **PWA Ready**: รองรับการติดตั้งเป็น Progressive Web App

### 🎮 การใช้งานที่หลากหลาย
- ปุ่มลัดสำหรับการเลือกฮีโร่บนมือถือ
- Touch feedback และ visual cues
- Auto-detection ของประเภทอุปกรณ์
- Dynamic sizing ตามขนาดหน้าจอ

## การติดตั้งและใช้งาน

### ข้อกำหนด
- Web browser ที่รองรับ ES6+ (Chrome, Firefox, Safari, Edge)
- ไม่ต้องติดตั้งอะไรเพิ่มเติม

### วิธีการใช้งาน
1. เปิดไฟล์ `index.html` ใน web browser
2. รอให้ข้อมูลฮีโร่โหลดเสร็จ
3. เลือกฮีโร่โดยคลิกที่รูปในแกลเลอรี่ แล้วคลิกที่ช่องทีม
4. ใช้ปุ่มกรองเพื่อค้นหาฮีโร่ที่ต้องการ
5. ดูการวิเคราะห์ทีมในส่วน Display

## ข้อมูลฮีโร่

ข้อมูลฮีโร่ถูกเก็บในไฟล์ `ROV.csv` ซึ่งประกอบด้วย:
- ชื่อฮีโร่
- Lane ที่เหมาะสม
- ประเภทดาเมจ
- ช่วงเวลาที่แข็งแกร่ง
- ค่า Damage, Durability, CC, Mobility
- Tier
- Ability พิเศษ
- ฮีโร่ที่ตอบโต้ได้

## การปรับแต่ง

### เพิ่มฮีโร่ใหม่
1. เพิ่มรูปภาพฮีโร่ในโฟลเดอร์ `asset/hero/`
2. เพิ่มข้อมูลฮีโร่ในไฟล์ `ROV.csv`

### ปรับแต่งการวิเคราะห์
แก้ไขฟังก์ชันใน `script.js`:
- `calculateTeamStats()` - คำนวณสถิติทีม
- `showTeamWarnings()` - แสดงคำเตือน
- `matchTypeFilter()` - กรองตามประเภท

### ปรับแต่ง Responsive Design
แก้ไข CSS custom properties ใน `style.css`:
```css
:root {
    --hero-size: 80px;        /* ขนาดฮีโร่ */
    --hero-gap: 10px;         /* ระยะห่างระหว่างฮีโร่ */
    --touch-target-size: 44px; /* ขนาดขั้นต่ำสำหรับการสัมผัส */
    --border-radius: 4px;     /* มุมโค้ง */
    --transition-duration: 0.2s; /* ความเร็วของ animation */
}
```

### Breakpoints ที่รองรับ
- **Large Desktop**: 1200px+ (ฮีโร่ขนาด 90px)
- **Desktop**: 992px-1199px (ฮีโร่ขนาด 85px)
- **Tablet Landscape**: 768px-991px (ฮีโร่ขนาด 75px)
- **Tablet Portrait**: 576px-767px (ฮีโร่ขนาด 70px)
- **Mobile Landscape**: 480px-575px (ฮีโร่ขนาด 65px)
- **Mobile Portrait**: 320px-479px (ฮีโร่ขนาด 60px)
